# Musical Esperança

Projeto gerenciado com Synkra AIOX + Claude Code.

## Estrutura

```
.aiox-core/     # Framework AIOX (não modificar)
.claude/        # Configuração Claude Code
docs/           # Stories, PRD, arquitetura
squads/         # Squads e agentes do projeto
scripts/        # Scripts utilitários
```

## Começar

Abra o projeto no Claude Code e ative o agente master:

```
/AIOX:agents:aiox-master
```
